# Dictionnaire compagnon des notes mystiques 

Ce dictionnaire tente de definir les concepts cles (a leur stade actuel de murissement)
mentionnes dans les notes mystiques.
Puisque les concepts definis ici sont a leur stade actuel de murissement, il 
est normal qu'ils contredisent parfois ce qu'il en est dit dans les premieres 
idees des notes mystiques.
